# Product Management REST API | Node.js & TypeScript
A robust and scalable RESTful API designed for product inventory management. This project emphasizes Type Safety, Automated Testing, and Standardized Documentation, following a modular architecture for clean code maintenance.

## 🛠️ Tech Stack
* *Runtime:* Node.js with TypeScript for end-to-end type safety.
* *Framework:* Express.js.
* *ORM:* Sequelize (Managing relational data models).
* *Testing:* Jest & Supertest (Unit and Integration testing).
* *Documentation:* Swagger / OpenAPI 3.0.

## 🚀 Key Features
* *Full CRUD Operations:* Specialized endpoints for the Product model.Modular
* *Routing:* Clean endpoint management via router.ts.
* *Automated QA:* Comprehensive test suite to ensure API reliability and prevents regressions ($100\%$ focus on core logic).
* *Live Documentation:* Interactive API explorer powered by Swagger.

## ⚙️ Quick Start
* *Clone & Install:* npm install
* *Database Setup:* Configure your .env for Sequelize.
* *Run Tests:* npm test
* *Development:* npm run dev
* *Docs:* Access /api-docs to view the Swagger UI.
